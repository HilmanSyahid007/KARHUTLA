# 🔥 KARHUTLA DETECTOR SITE

![Version](https://img.shields.io/badge/version-1.1.0-brightgreen)
![Status](https://img.shields.io/badge/status-active-blue)
![License](https://img.shields.io/badge/license-MIT-orange)

**KARHUTLA DETECTOR SITE** adalah platform web modern yang dirancang untuk memantau, melaporkan, dan mengedukasi masyarakat tentang Kebakaran Hutan dan Lahan (KARHUTLA) di Indonesia. Platform ini bertujuan untuk meningkatkan kesadaran publik dan mempercepat respon terhadap insiden kebakaran.

## ✨ Fitur Utama

- 🌓 **Dynamic Dark Mode**: Dukungan tema gelap/terang yang tersinkronisasi dan tersimpan di penyimpanan lokal.
- 📱 **Responsive Design**: Tampilan yang dioptimalkan untuk berbagai perangkat (Desktop, Tablet, dan Mobile).
- 📝 **Incident Reporting**: Formulir pelaporan insiden kebakaran hutan dengan fitur unggah bukti foto/video.
- 📺 **Educational Content**: Integrasi video edukasi dan sejarah mengenai dampak kebakaran hutan di Indonesia.
- 🎨 **Premium UI/UX**: Antarmuka modern dengan efek Glassmorphism dan animasi halus.

## 🛠️ Teknologi yang Digunakan

- **Core**: HTML5, Vanilla JavaScript (ES6+)
- **Styling**: Bootstrap 5, Custom CSS3 (CSS Variables, Flexbox, Grid)
- **Icons**: Bootstrap Icons
- **Fonts**: Google Fonts (Inter & Roboto Slab)

## 📂 Struktur Proyek

```text
KARHUTLA/
├── assets/          # Aset gambar dan ikon
├── css/             # File stylesheet (Bootstrap & Custom)
│   ├── styles.css   # Bootstrap Core
│   └── custom.css   # Desain Premium & Dark Mode
├── js/              # File logika JavaScript
│   ├── scripts.js   # Logika umum
│   └── theme-toggle.js # Logika Dark Mode
├── index.html       # Halaman Utama
├── report.html      # Halaman Pelaporan
├── history.html     # Halaman Sejarah & Dampak
├── login.html       # Halaman Masuk
└── register.html    # Halaman Pendaftaran
```
